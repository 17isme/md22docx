Step 1: Upload Style Template (.docx)

In your .docx template file, please do the following:

  1. Style various texts: Enter Markdown-style marker text like # Heading 1, ## Heading 2, > Quote, Body Text, etc., and set their respective styles.
  2. Style tables: Insert a table with any content into the document and set your desired borders, shading, and internal text styles.

The program will automatically recognize these elements and use their styles as the conversion standard.
